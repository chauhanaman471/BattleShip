Detailed Description

Class representing a ship with a bow and stern.

The Ship class contains two GridPosition objects representing the bow (front) and stern (back) of the ship. It includes methods for checking the validity of the ship's position, calculating the ship's length, and determining all the grid positions the ship occupies.
